# Portfolio Professionnel

Un portfolio web moderne et professionnel conçu avec HTML, Tailwind CSS et JavaScript.

## Caractéristiques

- ✅ Design moderne et épuré
- ✅ Responsive design (mobile, tablette, desktop)
- ✅ Tailwind CSS pour le styling
- ✅ Animations fluides et interactives
- ✅ Sections complètes : Header, Hero, About, Formation, Expériences, Compétences, Projets, Contact, Footer
- ✅ Menu mobile interactif
- ✅ Performance optimisée

## Sections

1. **Navigation** - Menu responsive avec liens vers toutes les sections
2. **Hero Section** - Présentation impactante avec boutons d'appel à l'action
3. **À propos** - Présentation personnelle et objectifs
4. **Formation** - Diplômes et certifications
5. **Expériences** - Historique professionnel détaillé
6. **Compétences** - Compétences techniques avec icônes
7. **Projets** - Portfolio de projets récents
8. **Contact** - Formulaire de contact et informations
9. **Footer** - Liens rapides et réseaux sociaux

## Technologies Utilisées

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (vanilla)
- Font Awesome (icônes)

## Installation

```bash
# Cloner le repository
git clone https://github.com/liqmidi/portfolio.git

# Naviguer vers le dossier
cd portfolio

# Ouvrir dans le navigateur
open index.html
```

## Déploiement

Le portfolio est déployé sur GitHub Pages à l'adresse suivante :
[https://liqmidi.github.io/portfolio](https://liqmidi.github.io/portfolio)

## Utilisation

Le portfolio est entièrement statique et ne nécessite aucune dépendance particulière. Il suffit d'ouvrir le fichier `index.html` dans un navigateur web.

### Personnalisation

Pour adapter le portfolio à votre profil, modifiez :
- Les informations personnelles
- Les sections formation et expériences
- Les projets affichés
- Les liens des réseaux sociaux
- Les couleurs (classes Tailwind)
